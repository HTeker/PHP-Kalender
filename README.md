# PHP Kalender

Dit was een schoolproject die ik in **mei 2014** heb gedaan.

## Demo
[Bekijk hier de demo](http://php-kalender.000webhostapp.com)

## Doel
De gebruiker een overzicht bieden van de geplande evenementen.

## Ontwikkelomgeving
HTML5, CSS3, PHP, SQL, TemplatePower

## Omschrijving
De evenementen die opgeslagen zijn in de database worden geïndiceerd in de kalender. Indien de gebruiker op een dag klikt worden de evenementen van die dag getoond in het notitieblokje.
